# Django Inventory Bare-Minimum Skeleton

This version is intentionally minimal so your group can build the actual pages yourselves.

## Setup

```bash
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Included

- `manage.py`
- Django project setup
- Inventory app
- Basic Inventory model
- Basic login/logout/signup routing
- Bare template files
- Basic API skeleton using Django REST Framework

## Group work needed

- Real login page design
- Real create-user page design
- Inventory dashboard layout
- API page styling/details
- CSS
- Extra fields/features
- Search/filtering
- Better validation
